A) Build SCSS-JS Source - ⇧

Requirements

  • NodeJS (Install)
  • Yarn (Install)

Scss is used for theme development. The source files are located in the assets/src directory. The compiled files are extracted to the dist directory. You need to use Yarn to compile. Theme dependencies are copied to the dist directory with the "yarn run build" command. Go to assets/src from console to compile.

Commands:

yarn install
yarn run build // Production
yarn run watch // Development
